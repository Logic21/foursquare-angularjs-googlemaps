### Gulp
    $ npm install gulp-cli -g
### Bower
    $ npm install -g bower

## Install and Run the project

```bash
# A. install the npm
$ npm install

# B. install the bower
$ bower install

# C. transpile sass
$ gulp frontend:sass

# D. run the project
$ node server.js

# E. visit the browser
$ http://0.0.0.0:3000
```
