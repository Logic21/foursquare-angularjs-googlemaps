### Gulp
    $ npm install gulp-cli -g
### Bower
    $ npm install -g bower

## Install and Run the project

```bash
# install the npm
$ npm install

# install the bower
$ bower install

# run the project
$ node server.js

# visit the browser
$ http://0.0.0.0:3000
```
